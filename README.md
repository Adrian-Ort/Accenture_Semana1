# Ejercicios Semana 1 – Módulo 1 y 2

Este repositorio contiene los ejercicios prácticos realizados durante la primera semana del curso, correspondientes a los módulos 1 y 2.

## Contenido

Durante esta semana se trabajó principalmente en:

- Fundamentos de Java
- Estructura de clases
- Métodos y tipos de datos
- Programación orientada a objetos básica
- Uso de listas y estructuras simples
- Ejercicios prácticos para reforzar lógica

Cada carpeta contiene ejemplos y prácticas realizadas en clase o como refuerzo personal.

## Objetivo

El objetivo de esta semana fue entender la base del lenguaje Java y cómo estructurar correctamente un proyecto simple antes de trabajar con frameworks como Spring Boot.

## Organización

- Cada ejercicio está separado por carpeta.
- Se mantiene una estructura clara para poder revisar fácilmente el progreso.
- El proyecto CRUD con Spring Boot se encuentra en la carpeta `Crud-Springboot`.

Este repositorio representa el avance progresivo en el aprendizaje de backend con Java.
